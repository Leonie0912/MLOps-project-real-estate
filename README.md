# MLOps-project-real-estate

![picture of a house](./images/building.jpeg)

March 9th, 2026 : 

Welcome to Léonie and Tommaso's final project for the MLOps class (DSBA M2). 

In this project, we build a tool for people who want to buy a property (a little bit like SeLoger). We also add a fraud detection feature and a "price is currently above / below average" feature. 

March 19th, 2026 : 

This project started our with this dataset : Demande de Valeurs Foncières (DVF) from data.gouv.fr website. This dataset is the record of all real estate transactions that happened in France in 2024. 
It is huge and messy, so it's important that we start this by cleaning the data. 

The first step is adding a .gitignore : "*.txt".
